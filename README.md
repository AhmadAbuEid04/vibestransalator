# Crossfeed

An Instagram companion concept that helps users discover posts from outside their own region with translation and AI-generated cultural context.

## Hackathon Goal

Build a lightweight browser extension or web prototype that:

- identifies posts from other regions
- translates captions into the user's language
- explains local slang or context
- helps users explore perspectives outside their own bubble

## Team Notes

- Start with a simple MVP
- Prefer a seeded dataset or mocked feed for demos
- Keep the focus on translation, region filtering, and context
